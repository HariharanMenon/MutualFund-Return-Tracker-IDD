# MutualFund-Return-Tracker-IDD
A mutual fund return tracking tool taking inspiration from Intent-Driven Development.

## Features
- Upload Indian Mutual Fund transactions and calculate XIRR (Extended Internal Rate of Return)
- Track returns on investment
- Built with IDD methodology

## Tech Stack
- Frontend: React + Vite
- Backend: Python
- Deployment: Render

## Quick Start

### Prerequisites
- Python 3.9+
- Node.js 18+
- Git

### Setup

**Backend:**
```bash
cd backend
python -m venv venv
source venv/Scripts/Activate.ps1  # Windows PowerShell
pip install -r requirements.txt
```

**Frontend:**
```bash
cd frontend
npm install
npm run dev
```
